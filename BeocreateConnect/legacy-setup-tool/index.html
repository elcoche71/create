Getting Started with SigmaStudio

Download SigmaStudio for free from Analog Devices website. Install it.

Get a starter project suitable for your loudspeakers from ausion website and open it in SigmaStudio.

Go to "Hardware Configuration" tab, right-click on the "TCPIP" block and select "Show TCPIP Settings".

Enter the IP address shown in the previous screen and click "Open Connection". If you see no errors, the connection has been made and you can close the window.

The Basics of Building and Testing a DSP Program

On the "Schematic" tab you will see all of the different blocks in the DSP program.

You can drag in new blocks from the toolbox on the left. To remove a block, select it and press the Delete key.

Click "Link Compile Download" in the toolbar to upload the program to the DSP memory. You should see "Active: Downloaded" in the lower right corner of the window. This means that the program has been uploaded.

When you change settings in existing blocks, you will hear the effect in real time, as long as the text in the lower right corner says "Active: Downloaded".

After adding or removing blocks, the text in the lower right corner will change to "Design Mode". This means that you will have to click "Link Compile Download" again to upload your changes and restore the real-time functionality.

Refer to the SigmaStudio help and wiki for more information on how to use the different DSP blocks.

Saving a Program to Permanent Memory

To make your program persist on the BeoCreate 4-Channel Amplifier when it is powered off, it needs to be written to permanent memory on the board.

Click "Link Compile Download" once more to make sure your changes have been updated to the DSP.

Go to "Hardware Configuration" tab and right-click on the "ADAU1451" block.

Under "Self-boot Memory" submenu, select "Write Latest Compilation through DSP".

Select OK in the window that opens – you don't need to change any of the settings.

Confirm that you want to erase and rewrite the memory, and wait for the process to complete. Do not unplug the sound system during this time.

Remember to also save the project file to your computer so you can make changes to it later, if you wish.

Remote Login Basics

Mac and Linux computers have a built-in SSH client accessible through the Terminal application. For Windows, PuTTY is a popular option that can be downloaded from the internet.

To open a connection, type "ssh pi@hostname", replacing "hostname" with the hostname or IP address from the previous screen. Press return.

If you are asked to confirm keys, type "yes" and press return, unless instructed otherwise.

Type your SSH password when asked and press return. It is normal to not see the password characters appear.

You should now be connected to the Raspberry Pi and you can run any commands on it.

To end the SSH session, type "exit" and press return.
